# Getting Started Guide — Common Setup Questions

This document addresses the most common onboarding and setup problems reported by new customers.

## Account not activated after signup

- Check spam/junk folder for the activation email.
- Activation links expire after 24 hours — resend from the login page ("Resend activation email").
- If resend does not work, support can manually activate the account from the admin panel.

## How to invite team members

1. Go to Settings > Team > Invite members.
2. Enter the email address and select a role.
3. An invite email is sent — valid for 7 days.
4. Invites can be resent or revoked from the same settings page.

## Connecting a third-party integration

For integrations (Slack, Zapier, GitHub, etc.):
1. Go to Settings > Integrations.
2. Click the integration name and follow the OAuth flow.
3. If the OAuth flow fails, ensure the customer's browser is not blocking pop-ups.

## Storage limits

- Free: 5 GB
- Pro: 100 GB
- Enterprise: Unlimited

## Setup guide is outdated

If a customer reports that the getting started guide refers to UI elements that no longer exist or steps that do not match the current product, log a documentation-gap ticket with the specific page URL and the discrepancy.
