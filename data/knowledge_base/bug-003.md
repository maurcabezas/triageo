# Browser Compatibility and Web App Issues

This guide covers common browser-specific issues and what information to collect from customers experiencing web app problems.

## Supported browsers

| Browser | Minimum version | Notes |
|---|---|---|
| Chrome | 110+ | Fully supported |
| Firefox | 110+ | Fully supported |
| Safari | 16.4+ | Fully supported |
| Edge | 110+ | Fully supported |
| Internet Explorer | Any | Not supported |

## Common browser issues

### Content not loading / blank screen

1. Hard refresh: Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac).
2. Clear browser cache and cookies.
3. Disable browser extensions (especially ad blockers) — try in incognito mode.
4. Check browser console for errors (F12 > Console) and share the error text.

### File upload not working

- Maximum file size is 500 MB.
- Ensure the browser has permission to access local files (not blocked by extension).
- Safari users: check that iCloud Private Relay is not interfering.

### Slow performance

- Try closing other browser tabs.
- Disable hardware acceleration if the app renders incorrectly on some GPUs.
